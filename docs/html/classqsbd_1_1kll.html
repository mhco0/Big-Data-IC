Detailed Description

template<class T>
class qsbd::kll< T >

A class to represent the Karnin-Lang-Liberty sketch.

The KLL summary provides a summary of data drawn from an ordered domain U. It is a comparison-based summary. The size of the summary is after any number of UPDATE and MERGE operations. It is more space- and time-efficient than the GK summary, but it is a randomized algorithm that may exceed the error bound with a small probability. The version of KLL described below assumes unweighted items.
