Detailed Description

A class to represent the Dyadic Count Sketch summary.

The DCS summary provides a compact summary of data drawn from an integer domain [U] = [0, 1, . . . , U − 1]. It takes (x,w) pairs as input where w can be an integer that is either positive or negative. But the multiplicity of x for any x in the data set must remain nonnegative when the summary is queried for it to be meaningful. It is a randomized algorithm. This summary takes space , and returns all ranks within εW error with constant probability, where W is the total weight of input integers.
