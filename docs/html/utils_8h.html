int qsbd::map_coord ( double  coord,
double  min_coord,
double  max_coord,
int  resolution 
)

This function maps some double coordenate into a int coordenate.

We subdivide the space based on the resolution given, imagine that we are subdividing a space that goes from min_coord = 0.0 to max_coord = 10.0, if we choose resolution equals to 2, we gonna have 2^2 = 4 possible spaces in int coordenates.

0.0-----—)2.5-----—)5.0-----—)7.5-----—)10.0

[0.0 .. 2.5) = 0 [2.5 .. 5.0) = 1 [5.0 .. 7.5) = 2 [7.5 .. 10.0) = 3

So, if we give a coordenate coord = 3.5 that coordenate will be mapped in a int coordenated equal to 1.

Parameters
coordThe double coordenate to be mapped
min_coordThe minimum coordenate that can appear in the space
max_coordThe maximum coordenate that can appear in the space
resolutionThe amout of subdivisions in the space
Returns
The coordenate coord mapped in a int space
Note
This can be use to map double coordenates into a quantile quadtree.
See also
quantile_quadtree

This function maps some int coordenate into a double coordenate.

We subdivide the space based on the resolution given, imagine that we are subdividing a space that goes from min_coord = 0.0 to max_coord = 10.0, if we choose resolution equals to 2, we gonna have 2^2 = 4 possible spaces in int coordenates.

[0.0 .. 2.5) = 0 [2.5 .. 5.0) = 1 [5.0 .. 7.5) = 2 [7.5 .. 10.0) = 3

So, if we give a coordenate coord = 1 that means that the double coordenated should be in range [2.5 .. 5.0), so we return the lower bound of this range.
