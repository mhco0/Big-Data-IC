Detailed Description

A class to estimate a frequency of a element in a stream.

It also summarizes a multiset v and answers point queries approximately. It has asymptotically the same guarantees as the CountMin Sketch, but with slightly worse hidden constant factors: Given parameters and , the summary uses space , and guarantees with probability at least 1−δ that any point query is answered with additive error at most . Alternatively, with space , it has error at most with probability at least . The benefit of the Count Sketch is that it produces an unbiased estimator.

This Count Sketch uses the L1 bound garantee.
