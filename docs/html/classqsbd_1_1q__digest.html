Detailed Description

A quantile sketch derived class.

The Q-Digest summary is based on maintaining a subset of simple counters of ranges, when the input is drawn from a fixed integer domain, and can be weighted. Each input is an pair where w must be positive. The summary takes space at most , which is fixed and independent of the input. Summaries can be easily merged together. The summary works by maintaining a tree-structure over the domain U: nodes in the tree correspond to subranges of U, and simple statistics are maintained for a small set of nodes.
