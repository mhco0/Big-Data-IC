class qsbd\+::quantile\+\_\+quadtree$<$ Obj\+Type $>$}
A quadtree that uses quantile sketchs in it\textquotesingle{}s nodes. 

A quadtree is a data structure that is used to subdivide a 2D-\/space into for region, each region in the subdivided space can also subdivide it\textquotesingle{}s space in four more regions. That can be use to make efficient queries and updates if the data has coordenates. A \mbox{\hyperlink{classqsbd_1_1quantile__quadtree}{quantile\+\_\+quadtree}} is a new data structure that simulates a quadtree behavier but also uses quantile\+\_\+sketchs in it\textquotesingle{}s nodes, that way, this structure is able to also anwer group statistics , such as the rank for a given element, the cdf for the data and it\textquotesingle{}s quantile. Remember that the quantile quadtree subdivides the space of this mode\+:
